## Application and File Dialog Boxes

These options open up file dialog boxes, or perform application-specific actions, such as a new browser tab, taking a browser out of full-screen, or opening developer tools within a browser. The highest value outcomes usually involve the File Open, File Save, and Print dialog boxes. 

## Test Cases

 - Ctrl+O (File Open Dialog Box)
 - Ctrl+S (File Save Dialog Box)
 - Ctrl+P (Print Dialog Box)
 - F1 (New Browser Tab)
 - F11 (Toggle Full-Screen Mode)
 - F12 (Browser Developer Tools)

## Next Steps

A large number of opportunities open up, related to File and Print dialog boxes, although in the case of a kiosk that is just a full-screen browser with limited restrictions, F11 to toggle full-screen can be helpful. 

## Ethical Note

Remember to only test on your own kiosks, or with permission such as in the scope of a bug bounty or penetration test, with good intentions of helping elevate kiosk security.
