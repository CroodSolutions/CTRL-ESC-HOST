## Basic OS Level Kiosk Escapes

These are the simple things that sometimes get overlooked, such as hitting Ctrl+Alt+Del and then launching Task Manager. 

## Test Cases

 - Ctrl+Alt+Del (Launch Task Manager)
 - Ctrl+Shift+Esc (Task Manager)
 - Alt+Tab (Change Active Window)
 - Alt+Esc (Change Active Window
 - Win+Tab (View All Active Windows)
 - Windows Key (Start Menu)
 - Ctrl+Esc (Start Menu)

## Next Steps

Task Manager allows users to launch new tasks, kill applications (if running in userland), and also allows for quite a bit of visibility into what is on the host (e.g., what EDR/AV/RMM, etc.)

## Ethical Note

Remember to only test on your own kiosks, or with permission such as in the scope of a bug bounty or penetration test, with good intentions of helping elevate kiosk security.
