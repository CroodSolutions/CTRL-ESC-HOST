## Tactic

These are a few basic shortcuts to Context and Control menus that could provide some escape possibilities.

## Test Cases

 - Alt+Spacebar (Opens context menu for the active window)
 - Alt+F4 (Closes the active window)
 - Shift+F10 (For some applications opens a context window)

## Next Steps

A common goal here could be to close or move aside the active window, which on very poorly locked down kiosks, is sometimes enough to escape kiosk mode. After this, what elements of the UI are available may vary, so often the next step is to try all of the other keyboard shortcuts (even if they did not work while the kiosk window was active).  

## Ethical Note

Remember to only test on your own kiosks, or with permission such as in the scope of a bug bounty or penetration test, with good intentions of helping elevate kiosk security.
