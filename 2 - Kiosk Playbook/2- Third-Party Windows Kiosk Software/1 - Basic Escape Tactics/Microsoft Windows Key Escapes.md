 ## Microsoft Windows Key Escapes

Microsoft Windows provides a wide range of keyboard shortcuts, using the Windows Key. It is important to test all of these, since individual 3rd party kiosk implementations will often lock down several, but miss a few that open up attack surface.  

## Test Cases

 - Win+R (Run Dialog Box)
 - Win+E (File Explorer)
 - Win+I (Settings)
 - Win+F (Feedback Hub)
 - Win+X (Quick Link Menu)
 - Win+S (Windows Search)
 - Win+Q (Windows Search)
 - Win+A (Accessibility)
 - Win+N (Notification Center)
 - Win+D (Desktop)
 - Win+M (Desktop)
 - Win+P (Projector)
 - Win+K (Cast Dialog Box)
 - Win+G (Game Bar / Xbox Social)
 - Win+V (Clipboard History)
 - Win+; (Emoji Dialog Box)
 - Win+. (Emoji Dialog Box)
 - Win+(+/-) (Magnifier) 

## Next Steps

The range of capabilities these open up are vast, but objectives usually include:

 - Settings Menu
 - Unlocked Web Browser
 - File Open / File Save Dialog Box
 - Win+X opens pathways to numerous features, including Terminal, File Explorer, Device Manager, Event Viewer, and more.  

## Ethical Note

Remember to only test on your own kiosks, or with permission such as in the scope of a bug bounty or penetration test, with good intentions of helping elevate kiosk security.
