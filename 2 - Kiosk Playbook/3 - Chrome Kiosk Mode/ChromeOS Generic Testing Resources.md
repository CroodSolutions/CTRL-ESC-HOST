## Introduction 

Here is a general listing of ChromeOS keyboard shortcuts that you can try to escape kiosk mode. Remember that there may be specific keyboard shortcuts for each specific application, which you may need to lookup based on the exact app you encounter. For a ChromeOS browser, look up normal Chrome Browser shortcuts also, although it seems like interesting attack surface on ChromeOS kiosks will be when a third party app is encountered (or a serious misconfiguration).

Some of these we have researched via articles and AI, but have never gotten to work. We are including the shortcuts we have not had success with, because they may be a fit for a misconfigured device you encounter, in a state we have not yet seen before.  

## Keyboard Shortcuts 

Note that most of these will not work, for a properly setup ChromeOS kiosk setup with Enterprise Enrollment, in a single app kiosk mode. That said, we test because things are not always setup correctly.

 - Esc+Refresh+Power (recovery mode)
 - Ctrl+Alt+R (powerwash)
 - Ctrl+Alt+T (crosh / chrome shell)
 - Ctrl+Alt+Refresh+D (demo)
 - Shift+Ctrl+Alt+D
 - Ctrl+Alt+D (demo mode)
 - Ctrl+Alt+E (enrollment options)
 - Ctrl+Shift+Q+Q (quick signout)
 - Ctrl+Alt+S (exit kiosk)
 - Ctrl+Alt+Shift+X (debug)
 - Ctrl+Alt+K (kiosk enrollment)
