While attacking a properly configured ChromeOS kiosk directly can be a daunting task, in particular with the time allotted during the normal scope of penetration testing, there could be interesting issues with any given app running on the kiosk.

In this example, the signage kiosk vendor AppSpace, simply allows you to unregister the kiosk, and register it again with your own (attacker controlled account).  

 - Ctrl+Alt+Shift+M
 - Select Unregister 
 - Setup the free trial of APPSPACE 
 - Click to Login or Register
 - Create content to display and update in APPSPACE 
 - Register the device to the new subscription
